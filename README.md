## 咩E-Ink Launcher

因为原作者已经不维护了，这是咩咩自用的修改版  
哔哩哔哩：郑羊羊  
基于E-Ink Launcher 0.1.8.6

### 修改内容
1. 尝试修复FTP功能，Wi-Fi开关等Bug
2. 支持暗色模式
3. 优化滑动翻页
4. 适配小屏幕  
5. 无需root的内存清理功能，写了个Golang项目 [fillRam](https://github.com/zanjie1999/fillRam) 实现  
6. 支持全按键操作，可以在没有触摸屏的设备使用
7. 点击底部时钟可以打开全屏时钟模式，点击时间返回，点击日期旋转屏幕
8. 屏幕方向设置，在启动时设置屏幕方向，避免因应用旋转导致桌面跟随应用方向
